## Usage

### Connections

```bash
# Add a new connection (tests it before saving)
meow connection add mydb postgresql://user:pass@localhost:5432/mydb

# List all saved connections
meow connection list

# Switch active connection
meow connection use mydb

# Show active connection info
meow connection info

# Remove a connection
meow connection remove mydb
```

### Explore

```bash
# List all schemas
meow schemas

# List tables in a schema (defaults to 'public')
meow tables
meow tables myschema

# Show rows from a table
meow rows users
meow rows users --schema myschema
meow rows users --limit 100
```

### Global Options

Every explore command supports:

```bash
--connection <name>   # Use a specific connection without switching active
```
